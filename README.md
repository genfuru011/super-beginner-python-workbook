44. 簡単な電卓アプリを作ろう 🧮💻

問題: ユーザーに2つの数字と演算子（+、-、*、/）を入力してもらい、計算結果を表示する簡単な電卓アプリを作成してください。

# 例:
def calculator():
    try:
        num1 = float(input("1つ目の数字を入力してください: "))
        op = input("演算子を入力してください (+, -, *, /): ")
        num2 = float(input("2つ目の数字を入力してください: "))
        
        if op == '+':
            return num1 + num2
        elif op == '-':
            return num1 - num2
        elif op == '*':
            return num1 * num2
        elif op == '/':
            return num1 / num2
        else:
            return "無効な演算子です。"
    except ZeroDivisionError:
        return "0で割ることはできません。"
    except ValueError:
        return "無効な入力です。"

result = calculator()
print(f"結果: {result}")
